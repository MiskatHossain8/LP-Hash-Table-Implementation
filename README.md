## ğŸ§  Concepts Used

- A Linear Probing Hash Table is a hashing technique that uses open addressing to handle collisions.
- When a collision occurs, the algorithm checks the next consecutive slots until an empty position is found.
- The probing sequence follows a linear pattern (index + 1, index + 2, etc.).
- Insertion, searching, and deletion all use the same probing method.
